# 📖 GhostVault Cheat Sheet

Welcome to GhostVault! This quick reference guide covers all the supported Markdown formatting inside the chat, as well as the advanced keyboard shortcuts you can use to master the application.

---

## 📝 Markdown Chat Formatting

GhostVault's chat supports lightweight, instant Markdown parsing so you can quickly format text and share code snippets.

| Style | Syntax | Example |
| :--- | :--- | :--- |
| **Bold** | `*text*` or `**text**` | *This is bold* |
| *Italic* | `_text_` | _This is italicized_ |
| `Inline Code` | `` `code` `` | `console.log('GhostVault')` |
| **Code Block** | \`\`\`language<br>code<br>\`\`\` | \`\`\`js<br>const a = 1;<br>\`\`\` |
| **Links** | Auto-detected | `https://google.com` becomes clickable |

**Pro Tip:** Code blocks are automatically syntax-highlighted and formatted with a beautiful dark theme.

## ⌨️ Global Keyboard Shortcuts

These shortcuts work globally across the entire GhostVault dashboard to keep your workflow lightning fast.

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl + V` <br>*(Cmd + V)* | **Paste / Upload** | Press anywhere on the dashboard to instantly upload an image from your clipboard to your personal vault. |
| `Esc` + `Esc` | **Panic Mode** | Quickly tap Escape twice to completely hide the UI. The screen goes pitch black. Click the tiny grey dot in the bottom right corner to restore. |
| `Esc` | **Close** | Closes any open modals, file viewers, or full-screen image previews. |

---

## 💬 Chat Specific Shortcuts

When you are inside a Tunnel / Chat Room:

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Enter` | **Send** | Instantly sends your typed message or attached files. |
| `Shift + Enter` <br>*(or Ctrl + Enter)* | **New Line** | Drops your cursor to a new line inside the chat input box without sending the message. |
| **Drag & Drop** | **Attach Files** | Drag any file or folder directly into the chat window to queue it up for sending. |
| `Ctrl + V` | **Paste Media** | Paste an image directly from your clipboard into the chat input bar to send it as an attachment. |
